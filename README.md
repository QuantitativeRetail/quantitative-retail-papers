# Quantitative Retail Papers

This repository contains the writing and publication materials for **Quantitative Retail**.

It includes:
- the foundational manifesto
- strategy audit papers
- bibliography files
- shared figures
- reusable LaTeX templates

## Current documents

- `manifesto/`  
  Foundational statement of mission, method, and evidentiary standards.

- `papers/ict-fvg-audit/`  
  First strategy audit paper on ICT Fair Value Gap.

## Writing philosophy

The goal of these papers is not marketing. The goal is to evaluate retail trading claims with clarity, realism, and methodological discipline.

## Planned sequence

1. Manifesto
2. ICT FVG strategy audit
3. Additional audits of common retail strategies
4. Broader papers on overfitting, regime instability, and benchmark comparison
